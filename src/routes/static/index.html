The Pandemic Monitor API is a service of access MongoDB database and provide data for mobile app created as prototype of the Projeto Integrador 8!

  • /version => Recurso para exibir a versão desta API.
  • /login/:email/:password => Recurso para validar o login no App, passando o email e asenha do usuário, necessário para responsáveis por alimentar os dados.
  • /users => Recurso para manipular a tabela "users" do banco de dados, é possível cadastrar (POST), alterar (PUT) e listar (GET) usuários.
    • /users/:email => Para pesquisar por email
  • /indicators => Recurso para manipular a coleção "indicators" do banco de dados, é possível cadastrar (POST), alterar (PUT) e listar (GET) valores dos indicadores da evolução da pandemia.
    • /indicators/:date => Para pesquisar por data.
    • /indicators/total => Para retornar os totais de casos, curados e óbitos.
    • /indicators/total/:date => Para retornar os totais de casos, curados e óbitos para a data especificada.
  • /informations => Recurso para manipular a coleção "informations" do banco de dados, é possível cadastrar (POST), alterar (PUT) e listar (GET) informações.
    • /informations/categories => Lista as categorias cadastradas
    • /informations/:category => Lista as informações de uma categoria
